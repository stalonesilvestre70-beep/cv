Профіль

AI-інженер та розробник автоматизацій із бекграундом в інженерній фізиці. Спеціалізуюся на побудові автономних AI-workflow, розгортанні self-hosted архітектури та інтеграції ШІ-агентів для бізнесу. Маю системний досвід у сфері автоматизації (n8n, Make, Zapier), збору даних (parsing) та побудови контент-пайплайнів. Експерт у налаштуванні конфіденційніших ШІ-рішень на базі локальних LLM.

Досвід роботи

  • Тестування та інтеграція ШІ-агентів (OpenClaw / Hermes): Конфігурація, тестування сценаріїв, верифікація відповідей та інтеграція локальних моделей і агентів у бізнес-процеси для автоматизації рутини й обробки запитів.
  • Проєктування n8n Workflows: Розробка self-hosted архітектури в Docker (PostgreSQL, Redis, Qdrant). Написання кастомного коду (JS/Python) в нодах для трансформації даних.
  • Впровадження локальних LLM: Інтеграція моделей через Ollama (Gemma, Qwen) у workflow компаній для забезпечення повної конфіденційності даних.
  • Повна автоматизація медіа: Створив систему автопостингу для Telegram-каналу: моніторинг джерел через API, рерайт/переклад за допомогою LLM. Пайплайн повністю замінив контент-менеджера.
  • Скрапінг та аналітика: Автоматичний парсинг Google Maps та сайтів через n8n із вивантаженням звітів у Google Sheets.
  • Побудова інтеграцій (Make / Zapier): Налаштування зв'язків між CRM-системами, сайтами, базами даних та Google Sheets через REST API та вебхуки.
  • Автоматизація маркетингу: Налаштування сценаріїв лідогенерації, автоматичного розподілу заявок та тригерних розсилок.
  • Автопостинг у месенджери: Створення систем для автоматичного збору та публікації контенту в Telegram-канали.
  • Infrastructure & Hardware: Проєктування, збірка та технічне адміністрування високопродуктивного заліза (NVIDIA RTX GPU). Оптимізація роботи серверного обладнання.
  • Web3 Operations: Конфігурація технічних ботів, управління активами та моніторинг стабільності мереж (Solana/Ethereum).
